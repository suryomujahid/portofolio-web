## Latar Belakang

Pada bulan Oktober 2022, Google Indonesia mengadakan event **Juara Android with Compose** sebagai kelanjutan dari event Juara Android sebelumnya. Kali ini, event fokus pada pembelajaran Jetpack Compose, sebuah toolkit modern untuk membangun UI pada aplikasi Android. Bahkan bagi mereka yang tidak memiliki latar belakang IT, event ini tetap bisa diikuti secara gratis.

Jetpack Compose memiliki banyak fitur unggulan yang memudahkan dan mempercepat proses pembuatan UI Android. Dibandingkan dengan XML, Compose menghasilkan kode yang jauh lebih sedikit, dan menghemat waktu dalam pengembangan dan perawatan kode.

## Pembelajaran

Saya mengikuti event ini dengan tujuan untuk membuat aplikasi saya sendiri dan meng-uploadnya ke Play Store. Saya ingin memperluas pengetahuan saya dalam pembuatan aplikasi Android native. Materi yang diberikan dalam event terdiri dari codelab dan quiz. Codelab adalah halaman tempat saya belajar materi-materi mengenai Compose, sedangkan quiz menilai pemahaman saya tentang materi yang telah diajarkan.

Materi dalam codelab dibagi menjadi dua bagian, yaitu [android basic compose](https://developer.android.com/courses/android-basics-compose/course) untuk pemula dan [android jetpack compose](https://developer.android.com/courses/jetpack-compose/course) untuk tingkat lanjut. Setelah menyelesaikan codelab dan quiz, saya mendapatkan badge yang tampil di profil Google Developer saya. Badge ini merupakan syarat untuk mendapatkan swag dari event Juara Android with Compose.
